# Rural Weaver API

FastAPI backend for Tejido Rural / Rural Weaver MVP.

## Stack

- FastAPI
- SQLAlchemy 2
- Alembic
- PostgreSQL
- JWT auth
- Docker Compose
- pytest

## Local development

```bash
cp .env.example .env
docker compose up --build
```

API:

```text
http://localhost:8000/api/health
```
